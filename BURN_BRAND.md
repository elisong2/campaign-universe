# Burn Studio — Brand Design Tokens

## Colors
- **Background (base):** #0E0E0E
- **Card background:** #161616 (no hard borders — use subtle shadow/elevation)
- **Primary text:** #F2EFE8 (Burn cream)
- **Secondary text:** #9A958C (warm muted gray)
- **Muted text:** #6B6B6B
- **Burn rust (primary accent):** #BF4723
- **Burn sage (healthy/positive):** #7C876F
- **Error/warning:** #BF4723 at higher opacity
- **Perplexity blue:** #5B8DEF
- **Card border (if needed):** #1E1E1E — prefer no borders, use elevation

## Typography
- **Font family:** -apple-system, BlinkMacSystemFont, 'Inter', sans-serif
- **Monospace (code/numbers in tables):** 'JetBrains Mono', ui-monospace, monospace
- **Headers:** Medium weight, sentence case (NOT ALL CAPS)
- **KPI numbers:** Bold, large, same sans-serif as body (not monospace)
- **Labels:** 10-12px, uppercase with letter-spacing ONLY for small tags/badges
- **Body:** 13-14px, regular weight

## Component Style
- **Card border-radius:** 16-20px (generously rounded)
- **Button/badge border-radius:** 10-12px
- **Progress bars:** pill-shaped (border-radius: 999px)
- **No hard borders on cards** — use background color difference + subtle box-shadow
- **Spacing:** generous — sections need breathing room

## Overall Feel
Premium internal tool. Warm, rounded, confident. Like Arc browser or Linear in dark mode, but with warm earth tones instead of cool blues. Bloomberg terminal data density meets modern design sensibility.
